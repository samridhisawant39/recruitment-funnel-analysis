# Recruitment Funnel Analysis & Hiring Insights

## Problem
Recruitment teams struggle to identify where candidates drop off in the hiring pipeline.

## Objective
Analyse hiring funnel performance and identify inefficiencies.

## Tools
- SQL
- Tableau
- Excel

## Dataset
Includes candidate data such as:
- Application date
- Interview date
- Offer date
- Source
- Experience level
- Status

## Key Analysis
- Candidate distribution by status, source, and experience
- Time-to-interview and time-to-offer
- Monthly hiring trends
- Funnel drop-off points

## Dashboard
[View Dashboar] (https://prod-uk-a.online.tableau.com/#/site/kalyanisawant3902-44ae2a1dcf/workbooks/3050730?:origin=card_share_link)

## Insights
- Majority drop-offs occur at interview stage
- LinkedIn and referrals are strongest sources
- Hiring process shows delay between interview and offer stages
